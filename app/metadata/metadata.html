Metadata

Information about our raw and summarized data types, as well as field-level metadata, can be found from the links below.

Name Description
Status and Intensity Metadata File Description Each row in this data type represents the status of one phenophase for one individual plant or animal species at a given site, on a single date and time.
Individual Phenometrics Metadata File Description Supports estimates of phenophase onset, duration and end, at the organismal level.
Site Phenometrics Metadata File Description Provides information on the mean first and the mean last positive observations for a given species and phenophase within a calendar year that are captured at a site within a user-defined interval.
Ancillary Data Provides ancillary information about our status and intensity field observation data, including details about the sites, individual plants, observers and protocols.
